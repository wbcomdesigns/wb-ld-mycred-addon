== Description ==
This plugin allows to assign myCred points on Learndash activities like course completion , lesson completion , topic completion , quiz completion.

**Note:** You will need to install the free [myCRED plugin](https://wordpress.org/plugins/mycred/ "myCRED")&trade; (version 1.7.9 or higher) to use the WBCOM Learndash MyCred Addon. You will also need the [LearnDash plugin](http://www.learndash.com "LearnDash") v1.4 or higher installed, available from LearnDash.com.

[Get the myCRED plugin](https://wordpress.org/plugins/mycred/ "myCRED").
[Get the LearnDash plugin](http://www.learndash.com/ "LearnDash").

[myCRED](https://wordpress.org/plugins/mycred/ "myCRED")&trade; lets you build a broad range of point related applications for your WordPress powered website.

**New ways to define achievements and give points for LearnDash activity**

myCRED supports some of the most popular plugins for WordPress like BuddyPress, WooCommerce, Jetpack, Contact Form 7 etc. To prevent to much cluttering in the admin area with settings, myCRED will only show features/settings for third-party plugins that are installed and enabled.

*   **Courses:**

   *   Complete any Course

*   **Lessons:**

   *   Complete any Lesson

*   **Quizzes:**

   *   Pass any Quiz

** In our next update we will cover up override options for any speciifc course , lesson and quiz and it will also allow to set separate points for each course, lesson, topic and quiz to override global settings. **

* Following Points will also be covered in our next update.

    *   Achieve a minimum percent grade on a specific Quiz
    *   Achieve a minimum percent grade on any Quiz
    *   Fail a Quiz (i.e. useful for rewarding the passing of a previously failed quiz)

== Installation Guide ==
1. Download the zip file and extract it.
2. Upload `wb-ld-mycred-addon` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Alternatively you can use WordPress Plugin installer from Dashboard->Plugins->Add New to add this plugin.

== User Guide ==
How can I assign points for Learndash activities?
To assign points on Laerndash Activities just drag drop the Learndash hook under points menu of myCred plugin and assign points there for each activities.

== Features ==
This plugin will add a hook for Learndash activities like course completion, lesson completion, quiz completion and will allow to assign points for them to user.
